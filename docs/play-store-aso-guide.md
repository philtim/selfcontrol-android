```
• Device Admin — Helps protect the active session from being interrupted.
• Notifications — Shows the focus session countdown in your notification bar.

Every permission is requested only when it is needed, and each one is explained before you grant it.
```

```
FocusTime is a distraction blocker for Android. It lets you block selected apps and websites during timed focus sessions.

2. Select a duration for your focus session.
3. Tap "Start Focus." Blocking services activate immediately.
4. If you open a blocked app, a blocking screen appears with a "Return Home" button. If you visit a blocked website, the DNS request is filtered and the site does not load.
5. When the session ends, all apps and websites are unblocked automatically.

• All data is stored locally on your device.
• No user account is required. No registration. No login.
• No analytics or tracking libraries are included.

• Usage Access — Required to detect which app is currently in the foreground, enabling app blocking.
• VPN — Required to create a local VPN service that filters DNS requests on the device for website blocking. No traffic is sent to external servers.
• Display Over Other Apps — Required to show the blocking overlay when a blocked app is opened.
• Device Admin — Used to help protect the active focus session.
• Notifications — Required to display the active session countdown notification.
```

## 3. Policy & Claim Safety Check

### 3.1 Risky Phrases & Compliant Alternatives

| # | Risky Phrase / Claim | Risk | Compliant Alternative |
|---|---------------------|------|----------------------|
| 1 | "Cannot be uninstalled" | Misleading; Android always allows uninstall via settings | "Helps protect the active session from being interrupted" |
| 2 | "Uninstall protection" | Implies blocking system-level behavior; policy risk | "Session protection via Device Admin" |
| 3 | "Prevents you from using apps" | Overpromises; user can always force-stop services | "Shows a blocking screen when you open a blocked app" |
| 4 | "Impossible to bypass" | Unverifiable; user can always revoke permissions | "Designed to keep you accountable" |
| 5 | "Locks your phone" | Suggests device locking, not app blocking | "Blocks selected apps during your focus session" |
| 6 | "VPN" (without context) | May trigger policy review; must clarify local-only | "Local-only VPN for on-device DNS filtering" |
| 7 | "Monitors your apps" | Sounds like surveillance | "Detects when a blocked app is opened" |
| 8 | "Tracks your usage" | Sounds like data collection | "Checks which app is in the foreground" |
| 9 | "Addiction cure" / "Cures phone addiction" | Medical claim; prohibited | "Helps you build better digital habits" |
| 10 | "Guaranteed focus" | Unverifiable outcome claim | "Designed to help you stay focused" |
| 11 | "100% effective" | Unverifiable | "Works across apps and websites" |
| 12 | "Blocks all distractions" | Overpromise; cannot block phone calls, etc. | "Block the apps and websites that distract you" |
| 13 | "Takes control of your device" | Implies malware-like behavior | "Puts you in control of your screen time" |
| 14 | "Hides apps" | Not what the app does; misleading | "Blocks access to selected apps during sessions" |

### 3.3 Required Disclosure Text Snippets

#### VPN Usage Disclosure (for Play Console Data Safety + Description)

```
VPN DISCLOSURE: FocusTime creates a local VPN connection on your device solely to
intercept DNS requests for the purpose of blocking websites you have selected.
This VPN operates entirely on-device. No network traffic is routed through
external servers. No browsing data is monitored, logged, or transmitted. The VPN
is active only during focus sessions and is terminated when the session ends.
```

#### Foreground Service Disclosure

```
FOREGROUND SERVICE DISCLOSURE: FocusTime uses foreground services to (1) monitor
which app is in the foreground so it can enforce app blocking, and (2) run the
local VPN for website blocking. These services display a persistent notification
showing your session countdown. Services run only during active focus sessions
and stop automatically when the session ends or the timer expires.
```

#### Overlay (SYSTEM_ALERT_WINDOW) Disclosure

```
OVERLAY DISCLOSURE: FocusTime uses the "Display Over Other Apps" permission to
show a full-screen blocking overlay when you attempt to open an app that is on
your blocklist during an active focus session. The overlay provides a "Return
Home" button and, if a master password is set, an "Emergency Override" option.
This overlay is only shown during active focus sessions.
```

#### Device Admin Disclosure

```
DEVICE ADMIN DISCLOSURE: FocusTime requests Device Administrator privileges to
help protect the active focus session. This prevents the app from being
uninstalled while a session is running. Device Admin can be deactivated at any
time through your device settings when no session is active.
```

#### QUERY_ALL_PACKAGES Disclosure

```
PACKAGE QUERY DISCLOSURE: FocusTime uses the QUERY_ALL_PACKAGES permission to
display a list of all installed apps so you can select which ones to block during
focus sessions. This information is used only for the app selection interface and
is not collected, stored beyond the blocklist, or transmitted.
```

#### Usage Access Disclosure

```
USAGE ACCESS DISCLOSURE: FocusTime uses the Usage Access permission to detect
which app is currently in the foreground. This is required to determine whether a
blocked app has been opened, so the blocking overlay can be displayed. Usage data
is checked in real time and is not stored, logged, or transmitted.
```

#### Text Overlay Rules (Screenshots)

- Maximum 5 words per headline overlay
- Headline font size: large enough to read at thumbnail size (~40–48pt on 1080px wide)
- Use high-contrast text (white text on dark background or dark text on light background)
- Never cover more than 30% of the screenshot area with text
- No misleading UI elements in overlays (no fake buttons, no fake ratings)
